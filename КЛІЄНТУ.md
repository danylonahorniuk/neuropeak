# Налаштування отримання заявок у Google Таблицю

Щоб заявки з сайту приходили у вашу Google Таблицю — виконайте ці кроки один раз. Займе ~5 хвилин.

---

## Крок 1 — Створіть Google Таблицю

1. Відкрийте [Google Sheets](https://sheets.google.com)
2. Створіть нову таблицю (натисніть **+**)
3. Назвіть її, наприклад: **NeuroPeak — Заявки**

---

## Крок 2 — Додайте скрипт

1. У таблиці відкрийте меню **Розширення → Apps Script**
2. Видаліть весь код що там є
3. Вставте цей код:

```javascript
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName("Заявки");

    if (!sheet) {
      sheet = ss.insertSheet("Заявки");
      var headers = sheet.getRange(1, 1, 1, 5);
      headers.setValues([["Дата", "Ім'я", "Телефон", "Email", "Коментар"]]);
      headers.setFontWeight("bold");
      headers.setBackground("#3ca366");
      headers.setFontColor("#ffffff");
      sheet.setFrozenRows(1);
      sheet.setColumnWidths(1, 5, 180);
    }

    sheet.appendRow([
      new Date().toLocaleString("uk-UA"),
      data.name    || "—",
      data.phone   || "—",
      data.email   || "—",
      data.comment || "—"
    ]);

    return ContentService
      .createTextOutput(JSON.stringify({ success: true }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ success: false, error: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet() {
  return ContentService
    .createTextOutput(JSON.stringify({ status: "ok" }))
    .setMimeType(ContentService.MimeType.JSON);
}
```

4. Натисніть **Зберегти** (іконка дискети або Ctrl+S)
5. Назвіть проект: **NeuroPeak**

---

## Крок 3 — Задеплойте скрипт

1. Натисніть кнопку **Розгорнути → Нове розгортання**
2. Тип: **Веб-додаток**
3. Налаштування:
   - **Виконувати як:** Я (ваш акаунт)
   - **Хто має доступ:** Усі
4. Натисніть **Розгорнути**
5. Google попросить дозвіл — натисніть **Авторизувати** та підтвердіть
6. Скопіюйте **URL веб-додатку** — він виглядає так:
   ```
   https://script.google.com/macros/s/XXXXXXXXXX/exec
   ```

---

## Крок 4 — Додайте URL у налаштування сайту

Передайте скопійований URL розробнику або самостійно:

### Якщо сайт на Vercel:
1. Зайдіть на [vercel.com](https://vercel.com) → ваш проект
2. **Settings → Environment Variables**
3. Додайте:
   - **Name:** `GOOGLE_SCRIPT_URL`
   - **Value:** ваш URL зі Кроку 3
4. Натисніть **Save** → зробіть новий деплой

### Якщо сайт запускається локально (для тестування):
Відкрийте файл `.env.local` у папці проекту і вставте URL:
```
GOOGLE_SCRIPT_URL=https://script.google.com/macros/s/XXXXXXXXXX/exec
```
