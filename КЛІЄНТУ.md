# Налаштування отримання заявок — NeuroPeak

Коли клієнт залишає заявку на сайті — ви отримуєте:
- 📊 Запис у Google Таблицю
- 📱 Повідомлення в Telegram

---

## ЧАСТИНА 1 — Google Таблиця

### Крок 1 — Створіть Google Таблицю

1. Відкрийте [Google Sheets](https://sheets.google.com)
2. Створіть нову таблицю
3. Назвіть її: **NeuroPeak — Заявки**

---

### Крок 2 — Створіть Telegram бота

> Це потрібно зробити до налаштування скрипту, щоб одразу вставити дані.

1. Відкрийте Telegram → знайдіть **@BotFather**
2. Напишіть `/newbot`
3. Придумайте назву бота (наприклад: `NeuroPeak Leads`)
4. Придумайте username бота (наприклад: `neuropeak_leads_bot`)
5. Скопіюйте **токен** — виглядає так: `7123456789:AAF...`

**Отримайте свій chat_id:**
1. Напишіть будь-що своєму новому боту (наприклад "привіт")
2. Відкрийте у браузері:
   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```
3. Знайдіть `"chat":{"id":XXXXXXXXX` — це і є ваш chat_id

---

### Крок 3 — Додайте скрипт у таблицю

1. У таблиці відкрийте **Розширення → Apps Script**
2. Видаліть весь код що там є
3. Вставте цей код, замінивши токен і chat_id на свої:

```javascript
// ── Налаштування ──────────────────────────────
var TELEGRAM_BOT_TOKEN = "ВСТАВИТИ_ТОКЕН_БОТА";
var TELEGRAM_CHAT_ID   = "ВСТАВИТИ_СВІЙ_CHAT_ID";
// ──────────────────────────────────────────────

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName("Заявки");

    if (!sheet) {
      sheet = ss.insertSheet("Заявки");
      var headers = sheet.getRange(1, 1, 1, 5);
      headers.setValues([["Дата", "Ім'я", "Телефон", "Email", "Коментар"]]);
      headers.setFontWeight("bold");
      headers.setBackground("#3ca366");
      headers.setFontColor("#ffffff");
      sheet.setFrozenRows(1);
      sheet.setColumnWidths(1, 5, 180);
    }

    sheet.appendRow([
      new Date().toLocaleString("uk-UA"),
      data.name    || "—",
      data.phone   || "—",
      data.email   || "—",
      data.comment || "—"
    ]);

    sendToTelegram(data);

    return ContentService
      .createTextOutput(JSON.stringify({ success: true }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ success: false, error: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function sendToTelegram(data) {
  if (!TELEGRAM_BOT_TOKEN || TELEGRAM_BOT_TOKEN === "ВСТАВИТИ_ТОКЕН_БОТА") return;

  var text =
    "🔔 Нова заявка — NeuroPeak\n\n" +
    "👤 Ім'я: "     + (data.name    || "—") + "\n" +
    "📞 Телефон: "  + (data.phone   || "—") + "\n" +
    "📧 Email: "    + (data.email   || "—") + "\n" +
    (data.comment ? "💬 Коментар: " + data.comment + "\n" : "") +
    "\n📅 " + new Date().toLocaleString("uk-UA");

  UrlFetchApp.fetch(
    "https://api.telegram.org/bot" + TELEGRAM_BOT_TOKEN + "/sendMessage",
    {
      method: "post",
      contentType: "application/json",
      payload: JSON.stringify({
        chat_id: TELEGRAM_CHAT_ID,
        text:    text
      })
    }
  );
}

function doGet() {
  return ContentService
    .createTextOutput(JSON.stringify({ status: "ok" }))
    .setMimeType(ContentService.MimeType.JSON);
}
```

4. Натисніть **Зберегти** (Ctrl+S)
5. Назвіть проект: **NeuroPeak**

---

### Крок 4 — Задеплойте скрипт

1. Натисніть **Розгорнути → Нове розгортання**
2. Тип: **Веб-додаток**
3. Налаштування:
   - **Виконувати як:** Я (ваш акаунт)
   - **Хто має доступ:** Усі
4. Натисніть **Розгорнути**
5. Google попросить дозвіл — натисніть **Advanced → Go to NeuroPeak (unsafe) → Allow**
6. Скопіюйте **URL веб-додатку**:
   ```
   https://script.google.com/macros/s/XXXXXXXXXX/exec
   ```

## ЧАСТИНА 2 — Підключення до сайту

### Якщо сайт на Vercel:
1. Зайдіть на [vercel.com](https://vercel.com) → ваш проект
2. **Settings → Environment Variables**
3. Додайте:
   - **Name:** `GOOGLE_SCRIPT_URL`
   - **Value:** URL зі Кроку 4
4. Натисніть **Save** → зробіть новий деплой

### Для локального тестування:
Відкрийте файл `.env.local` у папці проекту і вставте URL:
```
GOOGLE_SCRIPT_URL=https://script.google.com/macros/s/XXXXXXXXXX/exec
```
Потім перезапустіть сервер: `npm run dev`
